# Desktop-first, fully responsive (overrides buildmore mobile-primary)

buildmore-engineering treats mobile (390×844) as the primary viewport. For pos-pro,
per explicit project direction, we build **desktop-first** — the reporting Control Tower
and store drill-downs are dense, data-heavy views used primarily on desktop — while
keeping the app **fully responsive** so it remains usable on mobile. This reverses
buildmore's mobile-gate for this project only.

Status: accepted
